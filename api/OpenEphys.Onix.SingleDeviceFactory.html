SingleDeviceFactory

Abstract class that implements a DeviceFactory for a single device.

Properties

Property Type Description
DeviceAddress uint

Represents the address of the current device. Device addresses must be unique across the workflow.
DeviceName string

Represents the name of the current device. Device names must be unique across the workflow, as they are used to link input/output nodes to a specific configured device.
DeviceType Type

Represents the type of the current device.
