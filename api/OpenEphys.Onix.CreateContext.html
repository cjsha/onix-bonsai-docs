CreateContext

Creates the context in which all devices are contained. The driver to be used, and the index, are available here.

Properties

Property Type Description
Driver string

Represents the driver that is used to communicate with hardware. Possible values are "riffa", "test", and "ft600"
Index int

Represents the index specifying the physical slot occupied by hardware being controlled

Inputs/Outputs

Input/Output Type Description
Output IObservable<ContextTask>
