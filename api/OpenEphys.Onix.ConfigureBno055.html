ConfigureBno055

Sink node that configures a BNO055 device.

This is the minimal workflow to enable a headstage's orientation datastream:

Properties

Property Type Description
Enable bool

Specifies if the BNO055 device is enabled. If it is set to true, this device is available to stream data while the workflow is running.

Input/Output Type Description
Input IObservable<ContextTask>

Observable that holds a ContextTask
